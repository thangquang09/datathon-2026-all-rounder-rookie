2.2. CONTEST RULES

Entries will be evaluated by the judging panel based on the following main criteria:

Problem definition and analytical questions
Exploratory Data Analysis (EDA) and data understanding
Application and explanation of Machine Learning methods
Data visualization and ability to communicate insights
Practicality, ethics, and awareness of model limitations

The total score of each team will be aggregated based on the evaluation of the judging panel.


2.3. Core Values of Gridbreakers

1. System Thinking
Holistic/Systemic thinking

Gridbreakers always view data within the overall context of the business, understanding that a single number can reflect an entire chain of behaviors, decisions, and relationships within the operating system.

2. Courage to Break Patterns
Dare to challenge existing patterns

Gridbreakers do not passively accept existing assumptions. Instead, they constantly question, challenge what seems obvious, step out of their comfort zone, and dig deeper to discover new insights.

3. From Insight to Direction
Turning insights into decisions

Data only has value when it leads to action. Gridbreakers go beyond identifying trends—they transform insights into clear recommendations that help businesses optimize strategies and make better decisions.
